# stadszigt.github.io

Static website for Stadszigt, hosted via GitHub Pages.

## Structure

- `index.html`, `about.html`, `products.html`, `store.html` — site pages
- `assets/` — images and media
- `css/` — stylesheets
- `js/` — scripts

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000
